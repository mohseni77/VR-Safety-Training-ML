
#  VR-Based Industrial Safety Training with Machine Learning

## 📌 Project Overview

This project is a Virtual Reality (VR) based industrial safety training system designed to improve safety education and trainee evaluation using Machine Learning techniques.

The system simulates a realistic fire emergency scenario in an industrial environment, where users must follow correct safety procedures step-by-step. Their actions are recorded, analyzed, and evaluated to assess performance and safety behavior.

---

## 🎯 Research Objective

The main objective of this project is to enhance the evaluation of industrial safety training by combining:

* Immersive Virtual Reality simulation
* Behavioral data collection
* Machine Learning-based performance evaluation

The system aims to provide a more objective and accurate assessment compared to traditional training methods.

---

## 🛠 Technologies Used

* Unity Engine
* C# Programming
* Virtual Reality (VR)
* Data Logging (CSV)

---

## ⚙️ System Features

### 🧠 1. Behavior Tracking System

The system records each user action during the training process, including:

* Fire detection check
* Alarm activation
* Power shutdown
* Extinguisher selection
* Fire extinguishing outcome
* Exit behavior

---

### 🔁 2. Sequence Validation

A step-by-step safety procedure is enforced. The system detects:

* Correct sequence execution
* Procedural errors
* Behavioral deviations

---

### 📊 3. Data Collection System

The system automatically generates a structured dataset (CSV format) from user interactions, including:

* Action correctness
* Sequence accuracy
* Task completion time
* Final outcome

This dataset is designed for machine learning-based performance analysis.

---

### 🤖 4. Machine Learning Evaluation

User performance is evaluated using a scoring system based on collected behavioral data. The system returns:

* Final score
* Safety level classification
* Personalized feedback
